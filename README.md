# 🧠 NeuronIA — IA Colaborativa PT-BR

O **NeuronIA** é uma plataforma web experimental e moderna que combina inteligência artificial no lado do cliente (*client-side*) com uma infraestrutura na nuvem para criar uma experiência de IA colaborativa. O projecto foi desenvolvido num único ficheiro estático (`index.html`), o que o torna extremamente leve, rápido e fácil de alojar.

---

## 🚀 Funcionalidades Principal

- **Interface Futurista e Fluida:** Design responsivo com uma estética *dark mode* premium, utilizando as fontes *Space Grotesk* e *JetBrains Mono*.
- **Fundo Dinâmico Interativo:** Animação em tempo real baseada em redes neuronais gerada via HTML5 Canvas API.
- **Inteligência Artificial no Navegador:** Integração com o **TensorFlow.js** para executar modelos diretamente no ecossistema do utilizador.
- **Base de Dados em Tempo Real:** Conexão nativa com o **Supabase** para persistência de dados e funcionalidades colaborativas.
- **Segurança Reforçada:** Implementação rigorosa de políticas de segurança através de *Content Security Policy* (CSP) diretamente nas meta-tags.

---

## 🛠️ Tecnologias Utilizadas

O projecto foi construído sem a necessidade de *frameworks* complexos de frontend (como React ou Vue), recorrendo apenas a:

- **HTML5 & CSS3** (Variáveis CSS personalizadas e Layout Fluido)
- **JavaScript (ES6+)** (Manipulação de DOM assíncrona e Canvas)
- [TensorFlow.js](https://www.tensorflow.org/js) — Para processamento e execução de IA no browser.
- [Supabase](https://supabase.com/) — Como Backend-as-a-Service (BaaS) para gestão de dados.
